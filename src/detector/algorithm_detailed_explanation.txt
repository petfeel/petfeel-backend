# 반려견 행동 감지 시스템 상세 알고리즘 설명
==========================================

1. YOLO 모델을 통한 포즈 감지
---------------------------
1) 키포인트 감지 방식
   - 반려견의 몸체에서 24개의 주요 포인트 감지
   - 감지되는 키포인트:
     * 머리 부위: 코, 눈, 귀 등
     * 몸통: 어깨, 등, 골반
     * 다리: 앞다리와 뒷다리의 관절 포인트
     * 꼬리: 꼬리 시작점, 중간점, 끝점

2) 신뢰도 점수 처리
   - 각 키포인트마다 0~1 사이의 신뢰도 점수 계산
   - 0.7 이상: 높은 신뢰도로 감지된 포인트
   - 0.5~0.7: 중간 신뢰도, 참고용으로 사용
   - 0.5 미만: 낮은 신뢰도, 0으로 처리하여 무시

2. 특징점 정규화 프로세스
----------------------
1) 좌표 정규화
   - 기준점: 골반 중심점(14번 키포인트)
   - 모든 좌표를 골반 중심점 기준으로 변환
   - 계산식: (x - pelvis_x, y - pelvis_y)

2) 스케일 정규화
   - 기준: 양쪽 어깨 사이의 거리
   - 계산식: 정규화된_좌표 / 어깨_거리
   - 목적: 카메라와의 거리에 무관한 일관된 측정값 확보

3) 특징 벡터 생성
   - 각 키포인트당 2개의 값(x, y) 저장
   - 총 48개의 값을 가진 특징 벡터 생성
   - 신뢰도가 낮은 포인트는 (0, 0)으로 설정

3. 이상행동 감지 알고리즘
----------------------
1) 데이터 수집
   - 5프레임마다 1프레임 추출
   - 5개의 연속된 프레임으로 윈도우 구성
   - 각 윈도우는 약 0.8초의 행동 정보 포함

2) IsolationForest 분석
   - 원리: 정상 데이터와 다른 패턴을 가진 데이터 식별
   - 방법: 
     * 데이터를 랜덤하게 분할
     * 분할에 필요한 단계가 적을수록 이상치로 판단
   - 설정된 이상치 비율: 20%

3) LocalOutlierFactor 분석
   - 원리: 지역적 밀도 기반 이상치 탐지
   - 방법:
     * 각 데이터 포인트의 4개 이웃과 비교
     * 주변 데이터와의 밀도 차이 계산
   - 설정된 이상치 비율: 20%

4) 점수 통합 및 판단
   - 두 모델의 점수를 Z-점수로 정규화
   - 정규화된 점수의 평균 계산
   - 임계값 계산: 평균 - (표준편차 × 0.5)
   - 임계값보다 낮은 경우 이상행동으로 판단

4. 연속성 검사 및 행동 분류
------------------------
1) 연속성 검사
   - 5개의 연속된 윈도우 관찰 (약 4초)
   - 정상 행동 1회까지 허용
   - 4회 이상 이상행동 감지 시 경고 발생

2) 영상 저장 및 분석 트리거
   - 연속성 검사 통과 시 4초 영상 저장
   - 저장된 영상에 대해 Gemini AI 분석 요청
   - 분석 결과에 따라 0~4단계 분류

5. Gemini AI 분석 프로세스
------------------------
1) 입력 데이터 구성
   - 4초 분량의 행동 영상
   - 이상행동 감지된 프레임 수
   - 현재 상황 정보

2) 분석 항목
   - 현재 행동 상세 설명
   - 대처 방법 제시
   - 심각도 단계 평가 (0~4)

3) 단계 판정 기준
   - 행동의 지속성
   - 공격성/자해성 여부
   - 개입의 시급성

6. 실시간 처리 최적화
------------------
1) 프레임 처리 최적화
   - 5프레임 스킵으로 처리 부하 감소
   - 실제 처리 속도: 6fps
   - 버퍼 크기: 120프레임 (4초)

2) 메모리 관리
   - 특징점 버퍼: WINDOW_SIZE만큼만 유지
   - 프레임 버퍼: MAX_FRAMES만큼만 유지
   - 오래된 데이터 자동 삭제

3) 처리 효율화
   - 낮은 신뢰도 포인트 조기 필터링
   - 병렬 처리 가능한 구조
   - 메모리 사용량 최적화

7. 시스템 확장성
--------------
1) 모델 업데이트
   - YOLO 모델 가중치 파일 교체 가능
   - 이상치 탐지 파라미터 조정 가능
   - Gemini AI 프롬프트 수정 가능

2) 민감도 조정
   - STD_MULTIPLIER 조정으로 전체 민감도 제어
   - 이상치 비율 조정으로 개별 모델 민감도 제어
   - 허용 정상 행동 수 조정 가능

3) 처리 속도 조정
   - 프레임 스킵 수 조정
   - 윈도우 크기 조정
   - 버퍼 길이 조정
